# Jogo da velha
O objectivo é conseguir três círculos ou três xis ou bolinhas em linha, quer horizontal, vertical ou diagonal.

## Visualizar:
 - Primeiro clone o repositório:
    - ``` git clone git@github.com:T-abran/velha.git ```
 - Entre no repositório: 
    - ``` cd velha ```
 - Inicia o projeto:
    - ``` npm start ```
